 SkillConnect 🚀

SkillConnect is an exclusive student networking platform built for verified college students to connect, collaborate, grow together, and showcase their skills across different colleges. The platform creates a trusted student-only ecosystem where users can build meaningful academic and professional connections, collaborate on projects, and discover opportunities within the student community.

🌟 Features  
🔐 User Authentication & Secure Login

👤 User Profile Creation & Skill Showcase

🎓 Verified College Student Community

🤝 Collaborate with Students Across Colleges

📱 Fully Responsive Modern UI

⚡ Fast & Optimized Performance

🌐 Cross-College Networking Experience

📊 Student Dashboard & Profile Management

⚙️ Installation & Setup

1️⃣ Clone the Repository
git clone https://github.com/rahulkashyapp/skill-connect.git

2️⃣ Navigate to Project Folder
cd skill-connect

3️⃣ Install Dependencies
npm install

4️⃣ Run Development Server
npm run dev

Project will run on:

http://localhost:5173
